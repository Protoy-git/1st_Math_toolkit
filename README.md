## Math Python Toolkit

Python scripts demonstrating mathematical operations using NumPy.

### Topics Covered
- Matrix operations: inverse, determinant, transpose
- Statistics: mean, median, standard deviation
- Numerical Methods: bisection method, root finding

### Files
- `math_toolkit.py` — matrix operations and statistics
- `numerical_methods.py` — bisection method for root finding

### How to Run
python math_toolkit.py
<br>
python numerical_methods.py

### Requirements
pip install numpy
